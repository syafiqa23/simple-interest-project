# Simple Interest Calculator

This project is a simple calculator for computing **simple interest** based on user input.

## Features
- Calculates simple interest using principal, annual interest rate, and time period.
- Implemented in PowerShell script (`simple-interest.ps1`).

## Usage
1. Open PowerShell.
2. Run `.\simple-interest.ps1`
3. Enter principal, rate, and time.
4. View simple interest output.
